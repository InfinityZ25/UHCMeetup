# UHC-Meetup
The original NoobstersMC's 'UHC Meetup' plugin's source code. This minigame was built in a way intended to work only within the NoobstersMC server, changes are needed to be usable in other environments. Coded in native 1.8.8 and made compatible with 1.7 users using a modified Proxy.

# Installation
1.Clone the source code

2.Compile with desired modifications

2.Drag compiled plugin with dependencies(if still needed) onto the plugins folder.

3.Start your server.

4.Stop server and configure all the values
